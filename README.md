# Python Portfolio

This repository contains various Python projects and snippets aimed at demonstrating proficiency in Python programming.

## Structure

- **01_Python_Fundamentals**: Comprehensive operations on collections, logic flow, and file handling.
- **02_Object_Oriented_Programming**: OOP concepts from basic classes to advanced decorators and generators.
- **03_Data_Manipulation_Bits**: Data Science snippets using Numpy and Pandas.
- **04_Algorithms_and_Logic**: Algorithmic challenges and logic puzzles.
- **scripts_library**: Useful utility scripts.
